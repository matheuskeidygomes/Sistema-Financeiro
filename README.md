# Sistema-Financeiro

Sistema de cadastro de finanças pessoais, desenvolvido para fins de estudo e aprofundação no framework React.js.

# Tecnologias utilizadas  

- HTML/CSS  
- JavaScript  
- React.js (com hooks)  
- Redux (com persist store)  
- TailwindCSS
